## Argumentos por posicion

Sintaxis:

```csharp

string cadena = "Mundo";

Console.WriteLine("Hola {0}", cadena);

// salida: Hola Mundo

```
El {0} define la posición del argumento para reemplazarlo por el valor que tenga dicha variable.
